## Federation Layer (UN-type)
- advisory only
- no sovereignty
- issues recommendations and mediates treaty drafting

## Treaty Types
- Trade Treaty
- Non-Aggression Pact
- Mutual Defense (limited)
- Tech Sharing Accord
- Humanitarian Corridor Agreement

## Treaty Lifecycle
proposal -> negotiation -> ratification -> activation -> expiry/renewal
